## Stack technique

**Frontend & Apps**
| Technologie | Usage |
|---|---|
| HTML / CSS / JavaScript | Fondamentaux solides, sémantique, responsive, ES6+ |
| React / Next.js | Interfaces complexes, SSR/ISR, rendu haute performance |
| Vue.js / Nuxt | Applications réactives, composition API, state management |
| Svelte | Composants légers, compilation native, DX optimale |
| TypeScript | Typage fort, architecture scalable, code maintenable |
| SCSS / Tailwind CSS | Styling avancé, responsive design, prototypage rapide |

**Backend & Infrastructure**
| Technologie | Usage |
|---|---|
| Node.js | APIs performantes, services distribués, microservices |
| Rust | Performance native, sécurité mémoire, systèmes critiques |
| PostgreSQL / MySQL | Modélisation, optimisation, requêtes complexes |
| WebSocket | Communication temps réel, événements, streaming |
| Docker | Conteneurisation, environnements reproductibles |
| Electron | Applications desktop cross-platform, intégration native |
